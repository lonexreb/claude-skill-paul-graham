## What this repo contains

This repository contains:

1. **Links** to Paul Graham's essays at their canonical URLs on paulgraham.com.
2. **Short fair-use excerpts** (each ≤ 90 words) used for commentary,
   navigation, and to help readers locate the relevant passage in the
   original work.
3. **Our own synthesis, indexing, and thematic organization** — original
   prose written by the maintainer(s) of this repo to help Claude route
   user questions to the right essay.
4. **Structural metadata** (SKILL.md, manifest JSON) needed for the
   Claude Code skill to function.

This repository does **not** contain:

- Full essay text
- Any audio, video, or derivative formats of the essays
- Anything that would substitute for reading the originals

## Why we believe this is fair use

The use of short excerpts here is:

- **Transformative** — we add new structure (thematic routing, when-to-apply
  rules, cross-essay synthesis) so that Claude can match user questions to
  the right ideas. This is a different purpose from PG's original
  expressive intent.
- **Limited in amount** — each excerpt is ≤ 90 words; the rest of the
  reference material is our own original commentary.
- **Non-substitutive** — every reference points back to the canonical URL.
  Readers must visit paulgraham.com to read the full essay.
- **Non-commercial** — this repository is MIT-licensed open source with no
  paywall, ads, or revenue.
