# LinkedIn — Launch Post

**Recommended posting time:** Tuesday or Wednesday morning, 8–10am ET.
**Target audience:** founders, builders, engineers exploring entrepreneurship,
people in the Claude Code / AI builder community.

---

I open-sourced a Claude skill that turns Paul Graham's essays into an
on-demand advisor.

Every founder I know has read "How to Get Startup Ideas." Most have
forgotten 80% of it by the time they actually need it.

So I packaged ~40 of PG's essays on startups and high agency into a
Claude Code skill. Now when you ask Claude *"should I quit my job to
start this"*, it pulls the right essay, quotes the relevant lines, and
links you to the original.

What it does:
→ Auto-activates on startup, founder, and high-agency questions — you
don't have to name Paul Graham
→ Cites essays with canonical paulgraham.com URLs (no hallucinated quotes)
→ Organized by intent, not by title: ideas, growth, fundraising,
founders, product, mindset
→ MIT licensed, zero dependencies, drop into ~/.claude/skills/ and
you're done

Built with Claude Code. Took an afternoon. The interesting part wasn't
the code — it was deciding which 40 of PG's ~230 essays do the most work.

The full curated index covers the canon: How to Do Great Work, Schlep
Blindness, Do Things That Don't Scale, Relentlessly Resourceful, Default
Alive or Default Dead, Keep Your Identity Small, Life is Short, and the
rest.

Repo: github.com/lonexreb/claude-skill-paul-graham

If you build with Claude Code and you've ever wished PG's writing was
searchable by *intent* instead of by title — this is for you.
