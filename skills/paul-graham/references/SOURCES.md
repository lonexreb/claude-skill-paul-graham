# Sources

All essays referenced in this skill are by **Paul Graham**, published at
**https://www.paulgraham.com**. The canonical essay list is at
[paulgraham.com/articles.html](https://www.paulgraham.com/articles.html).

## Citation format

When the skill cites an essay, it uses this format:

> Paul Graham, *["Essay Title"](https://www.paulgraham.com/slug.html)*, YEAR.

## Accuracy and corrections

Years are approximate (PG sometimes revises essays without changing the
date stamp). URLs verified against the live site. Theses and excerpts
are summarized in good faith; if any are misleading, please open a
GitHub issue or PR.

## Refreshing this list

Run `scripts/fetch_essays.py` to regenerate the essay manifest from
paulgraham.com/articles.html. The script does not scrape essay bodies
— it only updates the URL/title index for future curation passes.
